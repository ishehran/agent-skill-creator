# Packaging Choices

Pick package and runtime model based on update, performance, and distribution needs.

## Common Options

- WinUI/WPF with .NET runtime
- Electron with bundled Chromium and Node
- Tauri with native webview and Rust backend

## Tradeoff Axes

- installer size
- memory footprint
- startup latency
- update mechanism complexity
- native API access

Document why the selected option fits your operational needs.
